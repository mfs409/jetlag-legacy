JetLag Tutorials

Below is a list of tutorials related to the JetLag game framework. After you've done the first two or three, the rest can probably be followed in any order. Please contact Mike Spear if you find any bugs or have any suggestions.

Getting Started
In this tutorial, we will discuss how to get started using JetLag. This tutorial doesn't involve much programming. It only focuses on getting your computer set up to start writing games. When you finish this tutorial, you will have a copy of the JetLag code that you are able to run. You will be able to use this as the starting point for learning about games, and also as the starting point for making your own game.
An Overview Of JetLag
This tutorial gives a high-level overview of JetLag. It briefly describes some of the main concepts that you will want to understand as you learn about game development, and then it explains how the default game works.
The Camera And Gravity
In JetLag, the camera decides what part of a level is visible at any time. The camera and gravity work together in interesting ways, which we explore in this tutorial.
Rigid Bodies and Physics
This tutorial discusses important issues related to rigid body types, physics properties, gravity, shapes, and tilt.
Styles of Movement
This tutorial relates the discussion of physics to the Movement field in the constructor for Actors.
Roles that Actors Play (1/2)
In JetLag, actors take on roles, like "Hero" or "Obstacle". This tutorial discusses the "Goodie", "Destination", and "Sensor" roles.
Roles that Actors Play (2/2)
This tutorial focuses on the "Obstacle" and "Enemy" roles.
Text And The Heads-Up Display
This tutorial discusses the ways that you can create text that appears in your game. It also introduces the heads-up display (HUD).
Graphical Assets
This tutorial begins a discussion about the different ways that you can control the appearance of an Actor.
Animations
This tutorial discusses animations.
Sound Effects
This tutorial introduces sound effects.
Transitions Among Stages
When you are designing a game, you'll quickly find that putting everything in one file is a bad idea. This tutorial presents one way of organizing your code, and also discusses how to transition between the different parts of your game. This tutorial also shows how to do background music.
Timer Events
Timers provide a way to make events happen in the future. This tutorial covers the basics of how to use them.
Keeping Score (1/2)
JetLag tracks a lot of different statistics, which you can use as part of the "score" in your game.
Keeping Score (2/2)
This is the second half of the tutorial about how JetLag keeps track of the "score" in your game.
Gesture Input
Mouse and touch inputs are among the most common ways of interacting with a game. This tutorial explores their use in JetLag.
